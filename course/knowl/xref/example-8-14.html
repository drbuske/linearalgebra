Example 5.2.5.

Consider the vector space \(\mathrm{Seq}_{\mathbb{R}}\) of infinite sequences of real numbers. For every \(k\in\mathbb{N}\text{,}\) let \(e^k\) be the sequence whose \(k\)th element is 1 and that is 0 everywhere else, i.e.,
\begin{gather*} e^0=(1,0,0,0,0,\ldots),\\ e^1=(0,1,0,0,0,\ldots),\\ e^2=(0,0,1,0,0,\ldots), \end{gather*}
and so on. Let \(S=\{e^k : k\in\mathbb{N}\}\text{.}\) Which of the following sequences are in \(\mathrm{span}(S)\text{?}\)
🔗
\begin{gather*} \text{(a)}\ f=(1,1,1,0,0,0,0,0,\ldots), \\ \text{(b)}\ g=(1,2,0,5,0,0,0,0,\ldots),\\ \text{(c)}\ h=(1,1,1,1,1,1,1,1,\ldots), \\ \text{(d)}\ m=(1,0,1,0,1,0,1,0,\ldots). \end{gather*}
🔗
(a) We have \(f\in\mathrm{span}(S)\text{,}\) because \(f=e^0+e^1+e^2\text{.}\)
🔗
(b) We have \(g\in\mathrm{span}(S)\text{,}\) because \(g=1e^0+2e^1+5e^3\text{.}\)
🔗
(c) The sequence \(h\) is not in \(\mathrm{span}(S)\text{,}\) because each element of \(\mathrm{span}(S)\) is, by definition, a linear combination of finitely many elements of \(S\text{.}\) No linear combinations of finitely many \(e^k\) can end in infinitely many ones. Note that we are not permitted to write an infinite sum such as \(e^0+e^1+e^2+\cdots\text{.}\) Such infinite sums are not defined in vector spaces.
🔗
(d) The sequence \(m\) is not in \(\mathrm{span}(S)\text{,}\) for the same reason. We would need to add infinitely many sequences of the form \(e^k\) to get a sequence that contains infinitely many non-zero elements. However, this is not permitted by the definition of span.
