Example 5.2.7.

Let \(S=\{x^2,(x+1)^2,(x+2)^2\}\text{.}\) We show that \(S\) is a spanning set for \(\mathbb{P}_2\text{,}\) the vector space of all polynomials of degree at most 2.
🔗
This is analogous to Example 5.2.3. Consider an arbitrary element \(p(x)=p_2x^2+p_1x+p_0\) of \(\mathbb{P}_2\text{.}\) We must show that \(p(x)\in\mathrm{span}(S)\text{,}\) i.e., that there exist \(a,b,c\in \mathbb{R}\) such that
\begin{equation*} p(x)=ax^2+b(x+1)^2+c(x+2)^2. \end{equation*}
🔗
We can equivalently rewrite this equation as
\begin{equation*} p_2x^2+p_1x+p_0=(a+b+c)x^2+(2b+4c)x+(b+4c), \end{equation*}
which yields the system of equations
\begin{equation*} \begin{aligned} a + b + c \amp = p_2 \\ 2b + 4c \amp = p_1 \\ b + 4c \amp = p_0 \end{aligned} \qquad \simeq \qquad \begin{bmatrix} 1 \amp 1 \amp 1 \amp p_2 \\ 0 \amp 2 \amp 4 \amp p_1 \\ 0 \amp 1 \amp 4 \amp p_0 \end{bmatrix} \qquad \simeq \qquad \begin{bmatrix} 1 \amp 1 \amp 1 \amp p_2 \\ 0 \amp 2 \amp 4 \amp p_1 \\ 0 \amp 0 \amp 4 \amp 2p_0 - p_1 \end{bmatrix}. \end{equation*}
🔗
Since the system has rank 3, it has a solution. Therefore \(p(x)\in\mathrm{span}(S)\text{.}\) Since \(p(x)\) was an arbitrary element of \(\mathbb{P}_2\text{,}\) it follows that \(S\) is a spanning set for \(\mathbb{P}_2\text{.}\)
