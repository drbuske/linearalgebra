Example 5.3.5.

Let \(V=\mathrm{Seq}_{\mathbb{R}}\text{,}\) the vector space of sequences of real numbers. Let
\begin{equation*} W=\{a\in\mathrm{Seq}_{\mathbb{R}}\mid \text{for all }n\ge 0,\ a_{n+2}=a_n+a_{n+1}\}. \end{equation*}
In other words, \(W\) is the set of all sequences satisfying the recurrence relation \(a_{n+2}=a_n+a_{n+1}\text{.}\) Then \(W\) is a subspace of \(V\text{.}\)
🔗
Proof: Before we prove that \(W\) is a subspace, let us first consider an example. The following sequences are elements of \(W\text{,}\) because they both satisfy the recurrence:
🔗
\begin{align*} a\amp =(1,1,2,3,5,8,13,21,\ldots),\\ b\amp =(1,3,4,7,11,18,29,47,\ldots). \end{align*}
🔗
Note that if we add these sequences, we get
\begin{equation*} a+b=(2,4,6,10,16,26,42,68,\ldots), \end{equation*}
which again satisfies the recurrence. Therefore, the set \(W\) is closed under the addition of these particular sequences \(a\) and \(b\text{.}\)
🔗
We now prove the properties in general.
🔗
  1. Let \(z\) be the zero sequence, defined by \(z_n=0\) for all \(n\text{.}\) Then \(z\) satisfies the recurrence relation, since for all \(n\ge 0\text{,}\) \(z_{n+2}=0=z_n+z_{n+1}\text{.}\) Therefore \(z\in W\text{.}\)
    🔗
  2. To show that \(W\) is closed under addition, consider any two sequences \(a,b\in W\text{,}\) and let \(c=a+b\text{.}\) Then for all \(n\ge 0\text{,}\)
    \begin{equation*} c_{n+2}=a_{n+2}+b_{n+2} =(a_n+a_{n+1})+(b_n+b_{n+1}) =(a_n+b_n)+(a_{n+1}+b_{n+1}) =c_n+c_{n+1}, \end{equation*}
    so \(c\) satisfies the recurrence. It follows that \(c\in W\text{.}\)
    🔗
  3. To show that \(W\) is closed under scalar multiplication, consider any \(k\in\mathbb{R}\) and \(a\in W\text{,}\) and let \(c=ka\text{.}\) Then for all \(n\ge 0\text{,}\)
    \begin{equation*} c_{n+2}=ka_{n+2}=k(a_n+a_{n+1})=ka_n+ka_{n+1}=c_n+c_{n+1}, \end{equation*}
    so \(c\) satisfies the recurrence. It follows that \(c\in W\text{.}\)
    🔗
