Example 5.2.9.

Determine whether the polynomials \(x^2\text{,}\) \(x^2+2x-1\text{,}\) and \(2x^2-x+3\) are linearly independent.
🔗
According to the definition of linear independence, we must solve the equation
\begin{equation*} ax^2+b(x^2+2x-1)+c(2x^2-x+3)=0. \end{equation*}
If there is a non-trivial solution, the polynomials are linearly dependent. If there is only the trivial solution, they are linearly independent.
🔗
We first rearrange the left-hand side to collect equal powers of \(x\text{:}\)
\begin{equation*} (a+b+2c)x^2+(2b-c)x+(3c-b)=0. \end{equation*}
This turns into a system of 3 equations in 3 variables:
\begin{equation*} \begin{aligned} a+b+2c \amp = 0 \\ 2b-c \amp = 0 \\ 3c-b \amp = 0 \end{aligned} \simeq \begin{bmatrix} 1 \amp 1 \amp 2 \amp 0 \\ 0 \amp 2 \amp -1 \amp 0 \\ 0 \amp -1 \amp 3 \amp 0 \end{bmatrix} \simeq \begin{bmatrix} 1 \amp 1 \amp 2 \amp 0 \\ 0 \amp 2 \amp -1 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \end{bmatrix}. \end{equation*}
🔗
Since the system has rank 3, there are no free variables. The only solution is \(a=b=c=0\text{,}\) and the polynomials are linearly independent.
