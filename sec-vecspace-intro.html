    Section 5.1 Vector spaces

    Earlier, we considered \(\mathbb{R}^n\text{,}\) the set of \(n\)-dimensional column vectors. We now introduce a more general concept of “vector”, as an element of an abstract vector space. Basically, vectors are entities that can be added and scaled. While some vectors look like lists of numbers (for example, column vectors, row vectors), other kinds of vectors don’t look like lists of numbers at all (for example, functions, polynomials). Part of the power of linear algebra comes from our ability to find vector spaces in many unexpected places.
    🔗
    Much of the content of this chapter will be a repetition of things we have already seen in the context of \(\mathbb{R}^n\text{.}\) For example, we will be talking about linear combinations, linear independence, spanning sets, bases, subspaces, linear transformations, and so on. We initially introduced these concepts in the context of the vector space \(\mathbb{R}^n\text{,}\) so that they would be easier to understand. We will now see that they in fact apply to all vector spaces.
    🔗

    Preview Activity 5.1.1.

    Subsection 5.1.1 Definition of vector spaces

    Definition 5.1.1. Vector space.

    Let \(K\) be a field. A vector space over \(K\) is a set \(V\) equipped with two operations of addition and scalar multiplication, such that the following properties hold:
    🔗
    • A1. Commutative law of addition: \(\uvec + \vvec = \vvec + \uvec\text{.}\)
      🔗
    • A2. Associative law of addition: \((\uvec + \vvec) + \wvec = \uvec + (\vvec + \wvec)\text{.}\)
      🔗
    • A3. The existence of an additive unit: there exists an element \(\zerovec \in V\) such that for all \(\uvec\text{,}\) \(\uvec + \zerovec = \uvec\text{.}\)
      🔗
    • A4. The law of additive inverses: \(\uvec + (-\uvec) = \zerovec\text{.}\)
      🔗
    • SM1. The distributive law over vector addition: \(k(\uvec + \vvec) = k\uvec + k\vvec\text{.}\)
      🔗
    • SM2. The distributive law over scalar addition: \((k + \ell)\uvec = k\uvec + \ell \uvec\text{.}\)
      🔗
    • SM3. The associative law for scalar multiplication: \(k(\ell \uvec) = (k\ell)\uvec\text{.}\)
      🔗
    • SM4. The rule for multiplication by one: \(1\uvec = \uvec\text{.}\)
      🔗
    🔗
    The above definition is concerned about two operations: vector addition, denoted by \(\vvec + \wvec\text{,}\) and scalar multiplication, denoted by \(k\vvec\) or sometimes \(k \cdot \vvec\text{.}\) In the law of additive inverses, we have written \(-\uvec\) for \((-1)\uvec\text{.}\)
    🔗
    Often, the scalars will be real numbers, but it is also possible to use scalars from a different field \(K\text{.}\) We also use the term \(K\)-vector space to refer to a vector space over a field \(K\text{.}\) When \(K = \mathbb{R}\text{,}\) we also speak of a real vector space, and when \(K = \mathbb{C}\text{,}\) we speak of a complex vector space. If the field is clear from the context, we often don’t mention it at all, and just speak of a “vector space”.
    🔗
    The elements of a vector space are called vectors. Our first example of a vector space is of course \(\mathbb{R}^n\text{.}\)
    🔗

    Example 5.1.2.

    The set \(\mathbb{R}^n\) of \(n\)-dimensional real column vectors, with the usual operations of vector addition and scalar multiplication, is a (real) vector space.
    🔗
    Proof: Properties (A1)–(A4) hold by our definition of vector addition, and properties (SM1)–(SM4) hold by our definition of vector-scalar multiplication.
    🔗
    🔗

    Example 5.1.3.

    The set \(\mathbb{C}^n\) of \(n\)-dimensional complex column vectors, with the usual operations of vector addition and scalar multiplication, is a (complex) vector space.
    🔗
    🔗
    We now consider some other examples of vector spaces.
    🔗

    Example 5.1.4.

    Let \(\mathbb{P}_2\) be the set of all polynomials of degree at most 2 with real coefficients, i.e., expressions of the form \(p(x) = ax^2 + bx + c\text{,}\) where \(a,b,c \in \mathbb{R}\text{.}\) Define addition and scalar multiplication of polynomials in the usual way:
    \begin{gather*} (ax^2 + bx + c) + (a'x^2 + b'x + c') = (a + a')x^2 + (b + b')x + (c + c')\\ k(ax^2 + bx + c) = kax^2 + kbx + kc. \end{gather*}
    Then \(\mathbb{P}_2\) is a vector space.
    🔗
    Proof: To show that \(\mathbb{P}_2\) is a vector space, we verify the 8 vector space axioms. Let \(p(x) = a_2x^2 + a_1x + a_0\text{,}\) \(q(x) = b_2x^2 + b_1x + b_0\text{,}\) and \(r(x) = c_2x^2 + c_1x + c_0\) be polynomials in \(\mathbb{P}_2\) and let \(k,\ell\) be scalars.
    🔗
    (A1) We prove the commutative law of addition.
    \begin{align*} p(x) + q(x) \amp = (a_2x^2 + a_1x + a_0) + (b_2x^2 + b_1x + b_0)\\ \amp = (a_2 + b_2)x^2 + (a_1 + b_1)x + (a_0 + b_0). \end{align*}
    Since addition in \(\mathbb{R}\) is commutative, this equals
    \begin{gather*} (b_2 + a_2)x^2 + (b_1 + a_1)x + (b_0 + a_0) = q(x) + p(x). \end{gather*}
    🔗
    (A2) We prove the associative law of addition.
    \begin{align*} (p(x) + q(x)) + r(x) \amp = ((a_2 + b_2)x^2 + (a_1 + b_1)x + (a_0 + b_0)) + (c_2x^2 + c_1x + c_0)\\ \amp = ((a_2 + b_2) + c_2)x^2 + ((a_1 + b_1) + c_1)x + ((a_0 + b_0) + c_0). \end{align*}
    By associativity in \(\mathbb{R}\text{,}\) this equals
    \begin{align*} (a_2 + (b_2 + c_2))x^2 + (a_1 + (b_1 + c_1))x + (a_0 + (b_0 + c_0)) \amp = p(x) + (q(x) + r(x)). \end{align*}
    🔗
    (A3) To prove the existence of an additive unit, let \(0(x) = 0x^2 + 0x + 0\text{,}\) the so‑called zero polynomial. Then
    \begin{align*} p(x) + 0(x) \amp = (a_2 + 0)x^2 + (a_1 + 0)x + (a_0 + 0) \\ \amp = p(x). \end{align*}
    🔗
    (A4) We prove the law of additive inverses.
    \begin{align*} p(x) + (-p(x)) \amp = (a_2x^2 + a_1x + a_0) + (-a_2x^2 - a_1x - a_0)\\ \amp = (a_2 - a_2)x^2 + (a_1 - a_1)x + (a_0 - a_0) \\ \amp = 0(x). \end{align*}
    🔗
    (SM1) We prove the distributive law over vector addition.
    \begin{align*} k(p(x) + q(x)) \amp = k((a_2 + b_2)x^2 + (a_1 + b_1)x + (a_0 + b_0))\\ \amp = (ka_2 + kb_2)x^2 + (ka_1 + kb_1)x + (ka_0 + kb_0)\\ \amp = kp(x) + kq(x). \end{align*}
    🔗
    (SM2) We prove the distributive law over scalar addition.
    \begin{align*} (k + \ell)p(x) \amp = (k + \ell)(a_2x^2 + a_1x + a_0)\\ \amp = ka_2x^2 + ka_1x + ka_0 + \ell a_2x^2 + \ell a_1x + \ell a_0\\ \amp = kp(x) + \ell p(x). \end{align*}
    🔗
    (SM3) We prove the associative law for scalar multiplication.
    \begin{align*} k(\ell p(x)) \amp = k(\ell a_2x^2 + \ell a_1x + \ell a_0)\\ \amp = (k\ell)a_2x^2 + (k\ell)a_1x + (k\ell)a_0 \\ \amp = (k\ell)p(x). \end{align*}
    🔗
    (SM4) Finally, we prove the rule for multiplication by one.
    \begin{align*} 1p(x) \amp = 1(a_2x^2 + a_1x + a_0)\\ \amp = a_2x^2 + a_1x + a_0\\ \amp = p(x). \end{align*}
    🔗
    Since the operations of addition and scalar multiplication on \(\mathbb{P}_2\) satisfy the 8 vector space axioms, \(\mathbb{P}_2\) is a vector space.
    🔗
    🔗

    Example 5.1.5.

    Let \(M_{m,n}(\mathbb{R})\) be the set of all \(m\times n\)-matrices with real entries, together with the usual operations of matrix addition and scalar multiplication. Then \(M_{m,n}(\mathbb{R})\) is a vector space.
    🔗
    Proof: The properties (A1)–(A4) hold by our definition of matrix addition, and the properties (SM1)–(SM4) hold by our definition of scalar-matrix multiplication.
    🔗
    🔗

    Example 5.1.6.

    Let \(V\) denote the set of \(2\times3\)-matrices. Let us define a non-standard addition in \(V\) by \(A\oplus B = A\) for all matrices \(A,B\in V\text{.}\) Let scalar multiplication in \(V\) be the usual scalar multiplication of matrices. We show that \(V\) is not a vector space.
    🔗
    In order to show that \(V\) is not a vector space, it suffices to find one of the 8 axioms that is not satisfied. We will begin by examining the axioms for addition until one is found which does not hold. In fact, for this example, the very first axiom fails.
    🔗
    Let
    \begin{equation*} A=\begin{pmatrix}1\amp 0 \amp 0\\0\amp 0 \amp 0\end{pmatrix}, B=\begin{pmatrix}0\amp 0\amp 0\\1\amp 0\amp 0\end{pmatrix}. \end{equation*}
    Then \(A\oplus B=A\) and \(B\oplus A=B\text{.}\) Since \(A\neq B\text{,}\) we have \(A\oplus B\neq B\oplus A\text{,}\) so property (A1) is false.
    🔗
    🔗

    Example 5.1.7.

    Let \(X\) be a nonempty set and define \(\mathrm{Func}_{X,\mathbb{R}}\) to be the set of functions defined on \(X\) and valued in \(\mathbb{R}\text{.}\) In other words, the elements of \(\mathrm{Func}_{X,\mathbb{R}}\) are functions \(f:X\to \mathbb{R}\text{.}\) The sum of two functions is defined by \((f+g)(x)=f(x)+g(x)\text{,}\) and scalar multiplication is defined by \((kf)(x)=k(f(x))\text{.}\) Then \(\mathrm{Func}_{X,\mathbb{R}}\) is a vector space.
    🔗
    To verify that \(\mathrm{Func}_{X,\mathbb{R}}\) is a vector space, we must prove the 8 axioms of vector spaces. Let \(f,g,h\) be functions in \(\mathrm{Func}_{X,\mathbb{R}}\text{,}\) and let \(k,\ell\) be scalars.
    🔗
    Recall that two functions \(f,g\) are equal if for all \(x\in X\text{,}\) we have \(f(x)=g(x)\text{.}\)
    🔗
    (A1) For all \(x\in X\text{,}\) \((f+g)(x)=f(x)+g(x)=g(x)+f(x)=(g+f)(x)\text{.}\) Therefore \(f+g=g+f\text{.}\)
    🔗
    (A2) For all \(x\in X\text{,}\) \(((f+g)+h)(x)=f(x)+(g(x)+h(x))=(f+(g+h))(x)\text{.}\) Therefore \((f+g)+h=f+(g+h)\text{.}\)
    🔗
    (A3) Let \(0\) denote the function defined by \(0(x)=0\text{.}\) Then \((f+0)(x)=f(x)\) for all \(x\text{,}\) so \(f+0=f\text{.}\)
    🔗
    (A4) Let \(-f\) be defined by \((-f)(x)=-f(x)\text{.}\) Then \((f+(-f))(x)=0\) for all \(x\text{,}\) so \(f+(-f)=0\text{.}\)
    🔗
    (SM1) For all \(x\text{,}\) \((k(f+g))(x)=k(f(x)+g(x))=(kf+kg)(x)\text{,}\) hence \(k(f+g)=kf+kg\text{.}\)
    🔗
    (SM2) \(((k+\ell)f)(x)=kf(x)+\ell f(x)\text{,}\) so \((k+\ell)f=kf+\ell f\text{.}\)
    🔗
    (SM3) \(((k\ell)f)(x)=k(\ell f(x))\text{,}\) so \((k\ell)f=k(\ell f)\text{.}\)
    🔗
    (SM4) \((1f)(x)=f(x)\) for all \(x\text{,}\) so \(1f=f\text{.}\)
    🔗
    It follows that \(\mathrm{Func}_{X,\mathbb{R}}\) satisfies all the required axioms and is a vector space.
    🔗
    🔗
    For the next two examples of vector spaces, we leave the proofs as an exercise.
    🔗

    Example 5.1.8.

    A sequence of real numbers, i.e. elements of \(\mathbb{R}\) is an infinite list \((a_0,a_1,a_2,a_3,\ldots)\text{,}\) where \(a_i\in \mathbb{R}\) for all \(i\text{.}\) We also use the notation \((a_i)_{i\in\mathbb{N}}\text{,}\) or occasionally \((a_i)\text{,}\) to denote such a sequence. Let \(\mathrm{Seq}_{\mathbb{R}}\) be the set of sequences of real numbers.
    🔗
    We add two sequences by adding their \(i\)th elements: \((a_i)_{i\in\mathbb{N}}+(b_i)_{i\in\mathbb{N}} =(a_i+b_i)_{i\in\mathbb{N}} \text{.}\) We scale a sequence by scaling each of its elements: \(k(a_i)_{i\in\mathbb{N}}=(ka_i)_{i\in\mathbb{N}} \text{.}\) Then \(\mathrm{Seq}_{\mathbb{R}}\) is a vector space.
    🔗

    Proposition 5.1.10.

    In any vector space, the following are true:
    🔗
    1. The additive unit is unique. In other words, whenever \(\uvec+\vvec=\uvec\text{,}\) then \(\vvec=\zerovec\text{.}\)
      🔗
    2. Additive inverses are unique. In other words, whenever \(\uvec+\vvec=\zerovec\text{,}\) then \(\vvec=-\uvec\text{.}\)
      🔗
    3. \(0\uvec=\zerovec\) for all vectors \(\uvec\text{.}\)
      🔗
    4. The following cancellation law holds: if \(\uvec+\wvec=\vvec+\wvec\text{,}\) then \(\uvec=\vvec\text{.}\)
      🔗
    🔗

    Proof.
    We prove the first three properties, and leave the last one as an exercise. Assume \(V\) is any vector space over a field \(K\text{.}\)
    🔗
    (a) Consider arbitrary vectors \(\uvec,\vvec\in V\) and assume \(\uvec+\vvec=\uvec\text{.}\) Applying the law (A1) (commutative law) to the left-hand side, we have \(\vvec+\uvec=\uvec\text{.}\) Adding \(-\uvec\) to both sides of the equation, we have \((\vvec+\uvec)+(-\uvec)=\uvec+(-\uvec)\text{.}\) Applying the law (A2) (associative law) to the left-hand side, we have \(\vvec+(\uvec+(-\uvec))=\uvec+(-\uvec)\text{.}\) Applying the law (A4) (additive inverse law) to both sides of the equation, we have \(\vvec+\zerovec=\zerovec\text{.}\) Applying the law (A3) (additive unit law) to the left-hand side, we have \(\vvec=\zerovec\text{.}\) This proves that whenever \(\uvec+\vvec=\uvec\text{,}\) then \(\vvec=\zerovec\text{,}\) or in other words, \(\vvec=\zerovec\) is the only element acting as an additive unit.
    🔗
    (b) Consider arbitrary vectors \(\uvec,\vvec\in V\) and assume \(\uvec+\vvec=\zerovec\text{.}\) Applying the law (A1) (commutative law) to the left-hand side, we have \(\vvec+\uvec=\zerovec\text{.}\) Adding \(-\uvec\) to both sides of the equation, we have \((\vvec+\uvec)+(-\uvec)=\zerovec+(-\uvec)\text{.}\) Applying the law (A2) (associative law) to the left-hand side, we have \(\vvec+(\uvec+(-\uvec))=\zerovec+(-\uvec)\text{.}\) Applying the law (A4) (additive inverse law) to the left-hand side, we have \(\vvec+\zerovec=\zerovec+(-\uvec)\text{.}\) Applying the law (A1) (commutative law) to the right-hand side, we have \(\vvec+\zerovec=-\uvec+\zerovec\text{.}\) Applying the law (A3) (additive unit law) to both sides of the equation, we have \(\vvec=-\uvec\text{.}\) This proves that whenever \(\uvec+\vvec=\zerovec\text{,}\) then \(\vvec=-\uvec\text{,}\) or in other words, \(\vvec=-\uvec\) is the only element acting as an additive inverse of \(\uvec\text{.}\)
    🔗
    (c) First, note that the scalar \(0\in K\) satisfies the property \(0+0=0\text{,}\) by property (A3) of the definition of a field. Now let \(\uvec\in V\) be any vector. Using the vector space law (SM2) (distributive law over scalar addition) and \(0+0=0\text{,}\) we have \(0\uvec+0\uvec=(0+0)\uvec=0\uvec\text{.}\) Next, we use a small trick: add \(-(0\uvec)\) to both sides of the equation. This gives \((0\uvec+0\uvec)+(-(0\uvec))=0\uvec+(-(0\uvec))\text{.}\) Applying the additional laws (A2), (A4), and (A3), we have \(0\uvec+(0\uvec+(-(0\uvec)))=0\uvec+(-(0\uvec))\text{,}\) \(0\uvec+\zerovec=\zerovec\text{,}\) and \(0\uvec=\zerovec\text{.}\) This proves that \(0\uvec=\zerovec\) holds for all vectors \(\uvec\text{,}\) as desired.
    🔗
    (d) This is left as an exercise.
    🔗

    Subsection 5.1.2 Summary

    The main purpose of this section was to investigate the definition of a vector space over a field through identification of the required operations and axioms.
    🔗
    • Standard examples of vector spaces include \(\mathbb{R}^n\text{,}\) polynomial spaces, matrix spaces, and spaces of functions.
      🔗
    • We can determine whether a given set with specified operations is a vector space by checking the vector space axioms. We can also identify examples of sets that fail to be vector spaces and explain which axioms are violated.
      🔗
    • The vector space axioms derive basic consequences such as the uniqueness of the zero vector and additive inverses.
      🔗

      Vector spaces are defined by a small number of basic ideas. Match each statement with the idea it illustrates.
      🔗
    • Adding two vectors in the space always produces another vector in the same space.
    • Closure under addition
    • There is a vector that leaves every vector unchanged when added to it.
    • Zero vector
    • For every vector, there is another vector that adds with it to give the zero vector.
    • Additive inverse
    • Multiplying a vector by a real number produces another vector in the space.
    • Closure under scalar multiplication
    • Linear combinations of vectors in the space stay in the space.
